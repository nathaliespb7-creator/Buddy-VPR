# Интеграция модуля «Математика ВПР 4 класс 2026»

## Для разработчиков

- **Задания с чертежами (№ 5, 10):** использовать HTML5 Canvas или SVG; ответ от клиента передавать как данные (например, base64 или JSON-описание линий).
- **Числовые ответы:** поддерживать форматы `1000`, `1 000`, `1.000` — валидатор нормализует пробелы и запятую.
- **Время:** поддерживать форматы `15:45`, `15 ч 45 мин`, `15 часов 45 минут` — валидатор принимает все варианты.
- **Partial credit:** для заданий 3, 8, 11 реализовано начисление 1 балла за верный ход решения при ошибке в ответе.
- **Калькулятор:** только для самопроверки (не для сдачи); в ВПР запрещён.
- **Логирование:** типичные арифметические ошибки из `common_mistakes` можно использовать для аналитики.

## Для контент-менеджеров

- Все задачи должны соответствовать кодификатору ФИОКО 2026.
- Для заданий 3, 8, 11 указывать несколько способов решения в `correctSolution`.
- Для заданий с изображениями указывать размеры в см и масштаб (`grid_cell_cm` и т.п.).
- Избегать избыточных данных для базового уровня.
- Числа в задачах — в диапазонах ВПР (до 1 000 000).

## Для тестировщиков

- Прогнать валидатор на 100+ вариантах (верные / неверные / частичные).
- Проверить работу на мобильных (особенно canvas для чертежей).
- Убедиться, что partial credit выставляется корректно для заданий 3, 8, 11.
- Проверить все форматы времени и чисел.
- Провести тест с детьми 9–10 лет на понимание формулировок.

## Чек-лист перед загрузкой

- [ ] Все 11 заданий покрыты примерами (целевой минимум 30–50 на задание).
- [ ] Валидаторы протестированы на 100+ вариантах ответов.
- [ ] Partial credit работает для заданий 3, 8, 11.
- [ ] Canvas/SVG реализованы для заданий 5, 10.
- [ ] Подсказки адаптированы под 3 методики (Школа России, Петерсон, Эльконин).
- [ ] Аудит-скрипт проходит без критических ошибок.
- [ ] Проведён тест с детьми (5–10 человек).

## Файлы модуля

- `client/src/data/mathVprGrade4/` — конфиг, примеры, подсказки, типы.
- `client/src/lib/validators/mathValidator.ts` — валидатор ответов.
